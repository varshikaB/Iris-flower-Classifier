# 🌸 Iris Flower Classifier

A machine learning project that classifies Iris flowers into one of three species — **Setosa**, **Versicolor**, or **Virginica** — based on flower measurements using the **K-Nearest Neighbors (KNN)** classification algorithm.

## 📌 Project Overview

The Iris dataset is one of the most widely used datasets in machine learning and serves as an excellent introduction to classification problems. This project demonstrates the complete machine learning workflow, including:

* Data loading and exploration
* Data preprocessing
* Train-test splitting
* Model training using KNN
* Model evaluation
* Prediction of flower species

## 📊 Dataset

The project uses the famous Iris dataset, which contains **150 samples** distributed across three species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

### Features

| Feature      | Description              |
| ------------ | ------------------------ |
| Sepal Length | Length of the sepal (cm) |
| Sepal Width  | Width of the sepal (cm)  |
| Petal Length | Length of the petal (cm) |
| Petal Width  | Width of the petal (cm)  |

Target Variable:

* Species
